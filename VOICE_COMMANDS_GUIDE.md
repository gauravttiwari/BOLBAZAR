# 🎤 Voice Commands Guide - BolBazar

## How to Use Voice Commands

1. **Enable Microphone**: Click the microphone icon in the navbar (purple button)
2. **Speak Clearly**: Say the command clearly in English
3. **Wait for Response**: The system will respond with voice feedback

## ⚠️ Important Notes

1. **Browser Support**: Voice commands work best in Chrome and Edge browsers
2. **Microphone Permission**: Allow microphone access when prompted
3. **Clear Speech**: Speak clearly and wait for the previous command to complete
4. **Product Numbers**: Check product card numbers before adding to cart
5. **Cart Required**: Some commands (like checkout) require items in cart

---

## 🎯 Tips for Best Results

- 🗣️ Speak at normal pace, not too fast or slow
- 🔇 Minimize background noise
- ⏸️ Wait 1-2 seconds between commands
- 🔄 If command doesn't work, try rephrasing
- 🎤 Keep microphone icon active (purple color)

---

## 🐛 Troubleshooting

**Voice not working?**
- Check if microphone icon is active
- Verify browser supports voice (Chrome/Edge)
- Check microphone permissions in browser settings
- Refresh the page and try again

**Command not recognized?**
- Ensure you're using exact command phrases
- Check if you're on the correct page (some commands are page-specific)
- Try speaking more clearly or louder
