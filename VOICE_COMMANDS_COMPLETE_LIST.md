## 📱 **Voice Command Input Format**

### For Search Queries
```
Pattern: [trigger] [value]
Examples:
- English: "search: laptop" or "find iphone"
- Hindi: "खोजो: किताब" or "सर्च जूते"
```

### For Form Fields
```
Pattern: [field]: [value]
Examples:
```
